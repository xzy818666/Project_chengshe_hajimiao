## 1. 游戏概述

**赛博功德银行** 是一款桌面金融模拟器。玩家在仙宫主界面敲击法器积累"功德"，并在多资产市场中投资、借贷、风控，最终结算本轮轮回。

**核心循环**：
1. 启动菜单选择难度 → 进入功德堂
2. 点击法器赚取功德
3. 善财司（存贷/业障管理）
4. 幻缘所（多资产交易/定投/套利）
5. 法器阁（购买/装备法器）
6. 结算或爆仓 → 存档下世损失 → 新一轮

**难度配置** (`StartMenu` / `GameConfig`)：
| 难度 | 初始功德 | 下世储备 | 通胀率 | 信用分 | 最大杠杆 |
|---|---|---|---|---|---|
| 简单 | 2000 | 2000 | 0.5% | 800 | 3× |
| 普通 | 1000 | 1000 | 1.0% | 500 | 2× |
| 困难 | 500 | 500 | 2.0% | 200 | 1.5× |

### W1 功德堂 (MeritHall)

**窗口特性**：
- 固定 16:9 宽屏比例（初始 1600×900，最小 800×450）
- 背景图：`modified_main_background_new.jpeg`（2560×1440 仙宫场景）
- `resizeEvent` 强制保持 16:9 + 重新缩放背景防平铺

**布局分区**：
- **顶部 HUD**（半透明深蓝灰底 + 淡金色边框）：
  - 日期、功德余额（金色）、资产市值（绿色）、净资产（青色）、业障（红色）
  - 今日点击、自动收益/秒、效率系数、通胀率/日
  - 下世储备 | 已消耗
- **四角建筑热区**（无文字透明按钮，覆盖整个建筑区域）：
  - 左上 **幻缘所**、右上 **善财司**
  - 左下 **法器阁**、右下 **成就**
  - hover 时门口显示光芒射出效果（`door_glow.png` 1:3 竖向素材 + `QGraphicsDropShadowEffect` 外发光）
- **正中云海**：法器点击热区（press/release 切换法器贴图）
- **底部莲花台**：莲台法器显示位置（电子烧香、AI 诵经机）

**法器贴图系统（双槽位）**：
- **主动槽（云上）**：始终装备一件，决定点击收益。press 切换 striked/unstriked：
  - 基础木鱼：`basic_muyu_unstriked.png` / `basic_muyu_striked.png`
  - 涡轮增压木鱼：`turbo_muyu_unstriked.png` / `turbo_muyu_striked.png`
  - 量子佛珠：`quantum_beads_unstriked.png` / `quantum_beads_striked.png`
- **辅助槽（莲台）**：可同时装备多件，只提供自动收益/特效，静态 Label 无交互：
  - 基础烧香：`basic_incense.png`（暂无独立贴图，临时复用 `electronic_incense.png`）
  - 电子烧香：`electronic_incense.png`
  - AI 诵经机：`ai_chanting.png`

**核心交互**：
- 点击云海 → `onInstrumentPressed()`（切换 striked 贴图）→ `onInstrumentReleased()`（恢复 unstriked + `Wallet::earn()`）→ 功德飘字动画 + 刷新 HUD
- 事件弹出通知：`QFrame` 从顶部滑入滑出，显示当前市场事件

### W2 善财司 (BankDialog)

**标签页 1：活期与定期**
- 活期余额与实时利息；定期存款列表（7天 6% / 30天 8% / 90天 12%），支持提前支取

**标签页 2：功德借贷**
- 债务总额、信用分（0–1000）、可借额度（`creditScore × 2`）
- 借款/还款；逾期债务按秒复利膨胀，超限时 10% 债务转化为业障
- **杠杆管理**：追加保证金、一键平仓

**标签页 3：业障状态**
- 业障值进度条；忏悔 Slider（1:1 消耗功德）
- 效率惩罚公式：`efficiency = base × max(0.5, 1 − yezhang/2000)`

**核心利率**：
- 活期 5%（年）；定期 7 天 6% / 30 天 8% / 90 天 12%
- 借贷日利率 1%

### W3 幻缘所 (ExchangeDialog)

**窗口特性**：
- 背景图：`stock_exchange.png`（古风交易所场景，1674×940 原图缩放至 1600×900）
- 窗口固定 1600×900，最小 800×450
- 控件采用绝对定位，四个面板对应背景图上的画框区域

**布局分区**：
- **最上面面板**（382,213～1299,281）：事件跑马灯 `eventTicker`
- **标签切换按钮**（376,285～594/594,285～794）："交易" / "资产配置" 两标签
- **中间左面板**（376,340～794,661）：
  - 交易模式：资产列表 `QListWidget`（众生法缘、善行福缘、因果私缘、轮回孽缘）
  - 资产配置模式：饼图 `QPieSeries` 显示持仓占比
- **中间右面板**（882,340～1315,660）：
  - 交易模式：价格标签 + K线图（`QLineSeries`，最近 100 tick）+ 持仓信息 + 交易输入框 + 杠杆选择 + 买卖按钮
  - 资产配置模式：总市值、风险评级 + **CAL 风险-收益图**
- **最下面面板**（361,720～1320,790）：智能推荐、套利检测（横排）+ 自动定投控件

**CAL 风险-收益图**（资产配置右面板）：
- `QScatterSeries` 蓝色散点：每个资产的（标准差 ×10⁴, 期望收益 ×10⁴）
- `QScatterSeries` 红色大圆点：当前实际持仓的加权组合点
- `QLineSeries` 红色虚线射线：资本市场线（CAL），从无风险利率点出发，穿过最大夏普比率资产，向右上方延伸至边界
- 资产名称标签紧贴散点显示

**核心功能**：
- 买入/卖出（支持杠杆 1×–3×）
- **智能配置建议**：`PortfolioAdvisor` 基于 Sharpe 比率加权推荐（已暴露真实的期望收益 `expectedReturn` 和标准差 `risk`）
- **套利提示**：`ArbitrageScanner` 实时扫描定价偏差
- **定投控制**：`AutoInvestor` 驱动，可设间隔/金额

**控件样式**：
- 列表/输入框/下拉框：背景 `transparent`，深褐文字 `#1A0F08`，直接融入背景画面板
- 按钮：木褐色底 `#8B5A2B` + 米白文字
- 图表：背景透明，坐标轴深色标签

**轮回孽缘特殊机制**：
- 卖出亏损时，优先从下世功德扣除，不足部分计入当世债务
- 维持率 < 30% 强制平仓

## 5. 资产定价体系

| 资产 | ID | 风险 | 定价机制 |
|---|---|---|---|
| 众生法缘 | `merit_index` | 中 | 几何布朗运动，基础漂移 0.02，波动 0.15 |
| 善行福缘 | `karma_bond` | 低 | 价格恒定为 1.0，持有即按秒产生固定收益（年化 4%） |
| 因果私缘 | `dharma_fund` | 中低 | `0.6 × merit_index_price + 0.4 × karma_bond_price` + 自身噪声 |
| 轮回孽缘 | `samsara_futures` | 高 | 独立随机游走，波动率 ×3，自带 2× 杠杆，维持率 < 30% 强制平仓 |

## 8. 关键代码约定

- **信号槽**：大量使用 Qt 信号槽进行界面-数据层通信（`Wallet` 信号 → `MeritHall` HUD 更新）
- **资产更新**：`main()` 中 `QTimer` 每 100ms tick 一次，驱动所有资产 `update(dt)`、市场事件、利息/通胀计算
- **自动收益**：`MeritHall` 中 `QTimer` 每秒触发 `updateAutoIncome()`，扣除维护费并发放自动收益
- **UI 生成**：`.ui` 文件由 Qt Designer 生成，运行时通过 `ui->setupUi(this)` 加载
- **资源路径**：所有图片通过 `resources.qrc` 打包，代码中使用 `":/images/xxx.png"` 访问

## 10. 构建与运行

```bash
/Users/qtx/Qt/6.11.0/macos/bin/qmake merit_bank.pro
make -j4
./merit_bank.app/Contents/MacOS/merit_bank
```

